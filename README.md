## About 
A Wikipedia search engine made in Nextjs using the <a href='https://www.mediawiki.org/wiki/API:Search'> MediaWiki Action API </a>. 
          
## Tech Stack
<ul>
          <li>Reactjs (Nextjs)</li>
          <li>Tailwindcss</li>
          <li>MediaWiki Action API</li>
          <li>Vercel</li>
 </ul>
